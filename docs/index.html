Install & run

  1. Download the installer (or the portable ZIP) above.
  2. Run it — installs per-user, no administrator rights needed.
  3. Launch Mulvakun, pick Tor or Proxies, then Connect.
  4. For Tor, optionally choose an exit country under the Connect button.
  5. Closing the window while connected minimizes to the system tray.
