Install & run

  1. Download the installer (or the portable ZIP) above.
  2. Run it — installs per-user, no administrator rights needed.
  3. Launch Mulvakun, pick Tor or Proxies, then Connect.
  4. For Tor, optionally choose an exit country under the Connect button.
  5. Closing the window while connected minimizes to the system tray.

Get Tor bridges (when Tor is blocked)

If your network blocks Tor, you need bridges — unlisted entry points. Ask the Tor Project for a few lines (obfs4, webtunnel or snowflake) using any of these, then paste them into Mulvakun.

Website bridges.torproject.org — open Advanced Options, choose obfs4 or webtunnel, solve the captcha, copy the lines.
Telegram bot @GetBridgesBot — send /start, then /bridges to get obfs4 bridges.
Email Mail bridges@torproject.org from Gmail/Riseup with body get transport obfs4. Reply contains the lines.

Add them in Mulvakun

  1. Open Settings (the ☰ button, top-right).
  2. Paste the bridge lines into “Tor bridges (one per line)” — one per line, exactly as given.
  3. Click Save, then choose Tor and press Connect.

A bridge line looks like: obfs4 12.34.56.78:443 <FINGERPRINT> cert=… iat-mode=0  ·  supported types: obfs4, webtunnel, snowflake, meek, conjure (and plain). Leave the field empty for a direct Tor connection.
